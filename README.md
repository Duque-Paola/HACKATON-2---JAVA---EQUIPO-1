# HACKATON-2---JAVA---EQUIPO-1
Hackaton basado en la creación de una agenda. Un contacto está definido por un nombre y un teléfono. Un contacto es igual a otro cuando sus nombres son iguales. Una agenda de contactos está formada por un conjunto de contactos. Se podrá crear de dos formas, indicándoles nosotros el tamaño o con un tamaño por defecto (10).
